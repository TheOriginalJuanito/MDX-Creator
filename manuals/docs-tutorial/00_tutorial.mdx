# Markdown Full Example

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Here's a complete example of its syntax.


<Container>
    <Row>
        <Col xs={12} md={6}>
            <div>
                ## 1. Headers

                Use `#` for headers:
                - `#` for H1
                - `##` for H2
                - `###` for H3
                - And so on...

                ### Example:
                # This is an H1
                ## This is an H2
                ### This is an H3
            </div>
        </Col>
        <Col xs={12} md={6}>
            <div>
                ## 2. Emphasis

                - *Italic*: Surround text with `*` or `_`.
                - **Bold**: Surround text with `**` or `__`.
                - ***Bold and Italic***: Use `***`.

                ### Example:
                *Italic*, **Bold**, ***Bold and Italic***
            </div>
        </Col>
    </Row>
</Container>

## 6. Blockquotes

Use `>` for blockquotes:

## 7. Tables

## 8. Horizontal Rules

Use `---`, `***`, or `___` for horizontal lines.

## 9. Task Lists

Use `- [ ]` for tasks:
